Who built this website?

This website was made by Emily Mould, Fiona Laird and Emily Shaw (aka team < br>illiant). We met 8 weeks ago when we started the CodeFirst: Girls 'Introduction to web development' course.

We chose to build this (imaginary) wedding website to showcase our skills.

During the intensive course, we learned HTML, CSS, JavaScript, Bootstrap, Github and principles of UX. We had 7 weeks to learn all of this and build a website! The course was 2-hour sessions every Tuesday evening over an 8-week period.
